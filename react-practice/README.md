# React practice

Small Vite + React app for learning and experiments.

## Scripts

- `npm run dev` — start dev server (default: [http://localhost:5173](http://localhost:5173))
- `npm run build` — production build to `dist/`
- `npm run preview` — preview the production build locally

## Where to edit

- `src/App.jsx` — main UI and examples
- `src/index.css` — global styles
- Add new components under `src/` and import them in `App.jsx`
